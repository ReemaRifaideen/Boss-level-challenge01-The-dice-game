## Overview
An interactive two-player dice game built with vanilla JavaScript, HTML, and CSS. Players compete by rolling virtual dice to accumulate points and win rounds.

## Getting Started

### Clone the Repository
```bash
git clone https://github.com/ReemaRifaideen/Boss-level-challenge01-The-dice-game.git
cd Boss-level-challenge01-The-dice-game
```

### Open the Game
Simply open `index.html` in your web browser to start playing!

## Features
- **Two-Player Gameplay** - Player 1 vs Player 2 competition
- **Real-time Scoreboard** - Track cumulative scores for both players
- **Unlimited Rolls** - Play as many rounds as you want
- **Smooth Animations** - Dice spin effect on each roll
- **Sound Effects** - Realistic tumbling dice sounds for immersive experience
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Keyboard Support** - Press SPACEBAR to roll or use the button
- **Modern UI** - Glass-morphism container with vibrant styling

## How to Play
1. Click **"🎲 Roll Dice"** button or press **SPACEBAR**
2. Watch the dice animate and reveal results
3. Player with the higher number wins that round and gains points
4. Click **"↻ Reset"** to clear scores and start fresh
5. Play unlimited rounds and track your wins!

## Game Rules
- Each player rolls one die (1-6)
- Higher roll wins the round
- Winner's score increases by 1
- Equal rolls = Draw (both players gain the point)
- No round limit - play forever!
